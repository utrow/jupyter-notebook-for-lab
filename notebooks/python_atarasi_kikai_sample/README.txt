●サンプルデータについて
サンプルデータには『Pyhonで動かして学ぶ！ あたらしい機械学習の教科書』の本文で解説したコードと実行結果のサンプルファイル（第2章から第9章）を用意しています。
サンプルファイルはJupyter Notebook形式です。
Jupyter Notebookのインストール方法は本文の第1章を参照してください。

・Jupyter Notebookへのサンプルファイルの追加方法
Jupyter Notebookを起動すると、［Files］タブの画面（Windows のエクスプローラーが起動）が開きます。
［Files］タブの画面にサンプルファイルをドラッグ＆ドロップして追加できます。
追加したら［Upload］ボタンをクリックします。
アップロードしたファイルをクリックすると編集画面が開きます。
［In］のボックスからコードをコピーできます。またファイル上で、実行結果も確認できます。

●本書のサンプルのテスト環境
　本書のサンプルは以下の環境で、問題なく動作することを確認しています。
　

OS：Windows 10

Anaconda：4.4.0
※本書で利用した「Anaconda3-4.4.0-Windows-x86_64.exe」はhttps://repo.continuum.io/archive/よりダウンロードできます。

Python：3.6.1
Jupyter Notebook：4.3.0
TensorFlow：1.3.0
Keras：2.0.8

●バージョンを指定したインストール方法
上記のTensorFlowおよびKerasのバージョンを指定してインストールする際は、本書のP.18、P.19のインストールの際に、以下のようにバージョン指定のコマンドを付けてインストールしてください。

pip install tensorflow==1.3.0

pip install keras==2.0.8
